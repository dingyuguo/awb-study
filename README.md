# 多光谱 AWB 研究全集

关于**多光谱自动白平衡（AWB）**与**局部 AWB（Local AWB）**的系统研究合集。

## 文档结构

| 文件 | 内容 |
|------|------|
| `多光谱AWB研究全集.html` / `index.html` | **带总目录的研究全集**（5 份文档合并，左侧目录可跳转） |
| `多光谱AWB研究报告.html` | 综合研究入口：背景、8 篇核心论文、华为专题、技术对比、数据集、未来方向 |
| `白平衡与光源概念辨析.html` | WB / CCM / 光源 / 白点 / 多光谱 概念统一框架 |
| `华为多光谱AWB路线图与论文详解.html` | 学术轨(IJCV2024/WACV2025) + 产业轨(红枫 1.0/2.0) 双轨解读 |
| `LocalAWB研究热点与待解难题.html` | 局部多光源 WB 原理、方案、八大难题、未来方向 |
| `CIC33论文详细讲解.html` | Spectricity CIC33 小白向逐句精讲 + 复现指南 + 引用论文分析 |

## 查看方式

直接用浏览器打开 `多光谱AWB研究全集.html`（或本仓库的 `index.html`）即可，无需任何依赖。
如需在线浏览，可在仓库 Settings → Pages 中启用（以 `index.html` 为入口）。

## 说明

- 本地论文 `CIC33.pdf`（Spectricity 等, CIC33 2025, *Enhancing Local Auto White Balance with Multispectral Images*）为**开源/开放获取论文**，已随本仓库发布，供学习研究使用。
- 由论文 PDF 提取的文本 `CIC33_fulltext.txt` **未**纳入仓库（仅本地留存）。
- 研究内容基于公开论文与公开技术资料整理，仅供参考与学习。
